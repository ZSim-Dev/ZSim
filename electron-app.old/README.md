# ZZZ Simulator

## 简述

绝区零伤害仿真软件客户端

## Project Setup

### Prepare

1. vscode
2. node >= 22.0
3. pnpm >= 10.0

### Select Package Manager

```bash
$ corepack install
```

### Install

```bash
$ pnpm i
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm run build:win

# For macOS
$ pnpm run build:mac

# For Linux
$ pnpm run build:linux
```
